# CodeAlpha Task 2 — Exploratory Data Analysis (EDA)

## Overview
Full EDA on a books dataset scraped from books.toscrape.com.
Covers data structure, statistics, distributions, correlations,
outlier detection, hypothesis testing and 6 visualisation charts.

## Files
- `task2_eda.py` — complete EDA script
- `books_scraped.csv` — dataset used for analysis

## Libraries Used
- pandas
- numpy
- matplotlib
- seaborn
- scipy

## How to Run
pip install pandas numpy matplotlib seaborn scipy
python task2_eda.py

## Key Findings
- 1000 books across 20 categories
- Average price: £13.97
- 87.8% books in stock
- Most books rated 3-4 stars (61%)
- Price and rating NOT significantly correlated
